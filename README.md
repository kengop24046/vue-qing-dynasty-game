# 我在大清当皇帝 - 纯前端文字养成游戏

基于 Vue3 + Vite + Pinia 开发的《穿越大清当皇帝》纯前端版，受到我在大清当皇帝HD启发，单机可玩，支持存档加密、导出导入。
游戏中你将扮演一个现代穿越到清朝的皇帝

##  声明
子嗣名字由AI输入提示词生成而成

## ✨ 核心功能
### 🏯 核心系统
- ✅ 30秒自动存档，本地加密存储，防篡改校验
- ✅ 存档导出/导入，JSON格式备份
- ✅ 游戏重置功能，一键清空进度
- ✅ 禁用页面自动刷新，防止数据丢失
- ✅ AES加密存档 + HmacSHA256哈希校验，修改存档直接失效

### 👑 皇帝核心玩法
- ✅ 金銮殿：国力、资源全局统计
- ✅ 上朝理政：随机政务事件，多选择分支
- ✅ 征粮征兵：充实国库，扩充兵力，冷却机制
- ✅ 万国纳贡：征服诸国解锁，定期领取贡品
- ✅ 国力等级：按国力解锁称号与功能，原版复刻经验体系
- ✅ VIP系统：20级VIP等级，兑换码充值，每日礼包，专属商城，专属后宫/大臣

### 🏮 后宫系统
- ✅ 完整妃嫔列表，详情查看
- ✅ 一键随机宠幸，单个宠幸
- ✅ 送礼提升好感度，等级晋升
- ✅ 宠幸生子，子嗣属性继承母亲资质
- ✅ 寻访解锁特殊佳人，副本/VIP专属妃嫔

### 👶 子嗣系统
- ✅ 皇子公主培养、教养成年
- ✅ 海量扩充命名字典，原版命名规则复刻
- ✅ 媒人系统：初始100个NPC子嗣，结婚后自动补充
- ✅ 子嗣联姻，属性继承，缘分羁绊加成
- ✅ 已婚子嗣详情查看，夫妻双方属性展示
- ✅ VIP刷新极品NPC子嗣，联姻外交绑定

### 📜 大臣系统
- ✅ 大臣列表、培养升级、详情查看
- ✅ 政治/军事双属性，职位晋升体系
- ✅ 寻访/副本/VIP解锁专属大臣

### ⚔️ 战斗玩法
- ✅ 普通副本：100个关卡，通关解锁专属后宫/大臣
- ✅ 征战副本：50个关卡，通关解锁纳贡国家
- ✅ 校场PK：玩家对战，排行榜，每日奖励，连胜机制
- ✅ 灵兽系统：神兽培养，出战加成，召唤系统

### 🔍 其他玩法
- ✅ 微服私访：寻访佳人、贤才、资源，多场景切换
- ✅ 助力礼包商城：金票购买成长礼包，原版复刻
- ✅ 周/月/年VIP特权体系，充值金额为唯一升级条件

## 🚀 快速开始
### 环境要求
- Node.js 18+
- npm/yarn/pnpm

### 开发运行
```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务
npm run dev

# 3. 访问 http://localhost:3000 即可开始游戏

### Docker部署

# 构建镜像
docker build -t vue-qing-dynasty-game .

# 运行容器
docker run -d -p 80:80 --name vue-qing-game qing-dynasty-game
